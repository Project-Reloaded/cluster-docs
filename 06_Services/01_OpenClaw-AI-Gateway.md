# OpenClaw AI Gateway

## Beschreibung
OpenClaw verbindet Chat, Werkzeuge und Infrastruktur-Automatisierung.

## Aufgaben
- Nachrichten verarbeiten
- Werkzeuge orchestrieren
- Persistente Notizen und Memory pflegen
- Operative Tasks im Homelab unterstuetzen

## Relevante Pfade
- Konfiguration: OpenClaw Gateway Config
- Workspace: `/root/.openclaw/workspace`
- Laufzeit auf VM 100
