# OpenClaw Memory Uebersicht

## Zweck
Diese Sektion sammelt dauerhaft relevantes Wissen aus dem laufenden Betrieb.

## Inhalte
- Entscheidungen
- Systembeziehungen
- Wiederkehrende Muster
- Verweise auf externe oder interne Doku

## Abgrenzung
Rohnotizen gehoeren in Tagesnotizen, verdichtetes Wissen in Memory.
